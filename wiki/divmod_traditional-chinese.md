# Python 的 divmod 函數：整數除法與餘數的完美結合

## 概述
`divmod` 是 Python 內建的一個函數，用於同時計算兩個數字的整數商和餘數。這個函數在需要進行整數除法的情況下，提供了一個方便且高效的方式來獲取結果。

## 文檔
### 功能
`divmod` 函數接受兩個參數，返回一個包含商和餘數的元組。這個函數的主要用途是在進行數學計算和算法設計時，快速獲取整數除法的結果。

### 使用方法
`divmod` 的語法如下：

```python
divmod(a, b)
```

- **參數**：
  - `a`：被除數，可以是整數或浮點數。
  - `b`：除數，必須是非零的整數或浮點數。

- **返回值**：
  - 返回一個元組 `(商, 餘數)`，其中 `商` 是整數商，`餘數` 是兩數相除後的餘數。

### 詳細描述
`divmod` 函數的運作方式是先計算整數商，然後計算餘數，這在一個函數調用中同時完成，使得代碼更加簡潔和高效。這個函數對於處理需要同時獲得商和餘數的情況特別有用，例如在數學計算、時間計算等場合。

## 例子
以下是一些 `divmod` 的基本用法示例：

```python
# 示例 1: 兩個正整數
result = divmod(10, 3)
print(result)  # 輸出: (3, 1)

# 示例 2: 負數與正數
result = divmod(-10, 3)
print(result)  # 輸出: (-4, 2)

# 示例 3: 正數與負數
result = divmod(10, -3)
print(result)  # 輸出: (-4, -2)

# 示例 4: 浮點數
result = divmod(10.5, 2)
print(result)  # 輸出: (5.0, 0.5)
```

## 解釋
在使用 `divmod` 時，需注意以下幾點：
- 除數 `b` 不能為零，否則會引發 `ZeroDivisionError`。
- 如果傳入浮點數，返回的商將是浮點數，這可能會造成精度問題，特別是在需要整數結果的情況下。
- 使用 `divmod` 可以比單獨使用 `//` 和 `%` 更高效，因為它只需進行一次除法運算。

## 總結
`divmod` 是 Python 中一個高效的函數，用於同時計算整數的商與餘數，能夠簡化代碼並提高計算效率。
