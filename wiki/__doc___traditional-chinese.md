# Python 的 __doc__ 屬性：深入了解 Python 文檔字串

## 摘要
`__doc__` 是 Python 中一個特殊的屬性，用於存儲函數、類別和模組的文檔字串，提供有關其功能和用法的簡要說明。

## 文檔
在 Python 中，`__doc__` 屬性是一個字符串，包含了用於描述模組、類別和函數的文檔字串。這些文檔字串通常位於定義的第一行，使用三重引號（""" 或 '''）來包裹，以便支持多行描述。透過 `__doc__`，開發者可以輕鬆地查看和理解代碼的功能，從而提高代碼的可讀性和可維護性。

### 目的
`__doc__` 屬性的主要目的是提供一種簡單的方式來存儲和訪問代碼的文檔，幫助開發者快速了解某個函數或類別的用途。

### 用法
要使用 `__doc__` 屬性，只需在定義函數、類別或模組時添加文檔字串，然後可以通過訪問 `.__doc__` 來檢索該文檔字串。

```python
def example_function():
    """這是一個範例函數的文檔字串。"""
    pass

print(example_function.__doc__)
```

## 範例
以下是使用 `__doc__` 的幾個基本範例：

### 函數範例
```python
def add(a, b):
    """返回兩個數字的和。"""
    return a + b

print(add.__doc__)  # 輸出：返回兩個數字的和。
```

### 類別範例
```python
class Dog:
    """這是一個表示狗的類別。"""

    def bark(self):
        """狗會叫。"""
        return "Woof!"

print(Dog.__doc__)        # 輸出：這是一個表示狗的類別。
print(Dog.bark.__doc__)   # 輸出：狗會叫。
```

### 模組範例
```python
"""這是一個示範模組的文檔字串。"""

def example():
    """示範函數。"""
    return "Hello, World!"

print(__doc__)  # 輸出：這是一個示範模組的文檔字串。
```

## 解釋
在使用 `__doc__` 時，有幾個常見的陷阱和注意事項：

1. **文檔字串的格式**：文檔字串必須位於函數、類別或模組的第一行，並且應使用三重引號包裹，以支持多行文本。
2. **可讀性**：即使可以省略文檔字串，也建議總是為函數和類別提供文檔，這可以顯著提高代碼的可讀性。
3. **自動化文檔生成**：許多自動化文檔工具（如 Sphinx）會提取 `__doc__` 屬性中的內容以生成文檔，因此保持文檔的準確性和完整性非常重要。

## 一句話總結
`__doc__` 屬性是 Python 中用於存儲和訪問函數、類別和模組文檔字串的特殊屬性，幫助開發者了解代碼的功能和用法。
