# Python中的abs函数：获取绝对值

## 概述
Python中的`abs()`函数用于返回一个数字的绝对值。无论输入是正数、负数还是零，`abs()`函数都会输出其非负值。

## 文档
`abs()`是Python内置函数之一，主要用于处理数值数据。其基本语法如下：

```python
abs(x)
```

### 参数
- `x`：可以是整数（int）、浮点数（float）或复数（complex）。 

### 返回值
- 返回输入值的绝对值。
- 对于复数，返回其模（即实部和虚部的平方和的平方根）。

### 使用场景
- 数学计算：在需要处理正负数时，获取绝对值是常见的需求。
- 数据分析：在分析数据集中处理负值和正值时，绝对值能够帮助规范化数据。

## 示例
以下是`abs()`函数的基本用法示例：

```python
# 整数的绝对值
print(abs(-5))  # 输出：5

# 浮点数的绝对值
print(abs(-3.14))  # 输出：3.14

# 复数的绝对值
print(abs(3 - 4j))  # 输出：5.0
```

## 说明
- 使用`abs()`时需注意，输入参数必须是数字类型，其他类型（如字符串、列表等）将导致`TypeError`。
- 对于复数，返回的值是浮点数，表示复数在复平面上的距离。

## 一句话总结
`abs()`函数返回数字的绝对值，是处理数值时非常实用的工具。
