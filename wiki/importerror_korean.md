# 파이썬 ImportError: 모듈 가져오기 오류 해결 가이드

## 개요
`ImportError`는 파이썬에서 모듈이나 패키지를 가져오려고 할 때 발생하는 오류로, 주로 모듈이 존재하지 않거나 잘못된 경로에서 가져오려 할 때 나타납니다.

## 문서
`ImportError`는 파이썬의 표준 오류 중 하나로, 특정 모듈이나 패키지를 찾을 수 없거나 로드할 수 없을 때 발생합니다. 이 오류는 코드 실행 중에 프로그램이 필요한 모듈을 찾지 못할 경우, 즉시 프로그램 실행을 중단시킵니다. 

### 목적
대부분의 경우, `ImportError`는 프로그래머가 잘못된 모듈 이름을 사용하거나, 모듈이 설치되지 않았거나, 다른 경로에서 잘못된 버전을 가져오려고 할 때 발생합니다. 이를 통해 개발자는 코드에서 발생할 수 있는 문제를 빠르게 파악하고 수정할 수 있습니다.

### 사용법
`ImportError`는 다음과 같은 상황에서 발생합니다:
- 존재하지 않는 모듈을 가져오려고 할 때.
- 모듈이 있지만, 해당 모듈 내의 특정 클래스를 찾을 수 없을 때.
- 환경 변수나 경로 설정이 잘못되어 모듈을 찾을 수 없을 때.

## 예제
### 예제 1: 없는 모듈 가져오기
```python
import non_existent_module
```
위 코드를 실행하면 `ImportError: No module named 'non_existent_module'` 오류가 발생합니다.

### 예제 2: 잘못된 클래스 가져오기
```python
from math import non_existent_function
```
이 경우, `ImportError: cannot import name 'non_existent_function'` 오류가 발생합니다.

### 예제 3: 잘못된 경로
```python
import my_module  # my_module이 현재 작업 디렉토리에 없을 경우
```
`ImportError`가 발생하며, 이는 해당 모듈이 현재 경로에 없음을 의미합니다.

## 설명
`ImportError`를 처리할 때는 다음과 같은 일반적인 실수나 주의사항이 있습니다:
- **모듈 이름 오타**: 모듈 이름을 잘못 입력하면 오류가 발생합니다.
- **경로 설정 문제**: PYTHONPATH 환경 변수가 올바르게 설정되지 않으면, 파이썬 인터프리터가 모듈을 찾지 못할 수 있습니다.
- **모듈 설치 여부**: 필요한 모듈이 설치되어 있는지 확인해야 합니다. `pip install` 명령어를 사용하여 필요한 패키지를 설치할 수 있습니다.
- **버전 호환성**: 특정 모듈이나 패키지가 사용하는 파이썬 버전에 따라 다를 수 있으므로, 호환성을 체크하는 것이 중요합니다.

## 한 줄 요약
`ImportError`는 파이썬에서 모듈을 가져오지 못할 때 발생하는 오류로, 주로 모듈의 이름이나 경로 설정 문제로 인해 발생합니다.
