# PythonのIndentationError：インデントエラーの理解と対処法

## 概要
Pythonにおける`IndentationError`は、コードのインデントが正しくない場合に発生するエラーです。このエラーは、Pythonの文法においてインデントが重要な役割を果たすため、特に注意が必要です。

## ドキュメンテーション
### 目的
`IndentationError`は、プログラム内で適切なインデントが守られていないときに発生し、主に制御構造（if文、forループ、関数定義など）のブロックが正しく認識されないことによって引き起こされます。

### 使用法
Pythonでは、コードのインデントはコードの構造を示すために使用されます。インデントは通常、スペースまたはタブを用いて行いますが、混在させることは避けるべきです。インデントの深さに応じて、どのコードブロックに属するかが決まります。

### 詳細
- **エラーの発生**: `IndentationError`は、以下のような状況で発生します。
  - インデントが欠如しているコード行。
  - インデントの深さが不一致な場合。
  - 不適切な文字でインデントされた場合（タブとスペースの混在）。

エラーメッセージには「expected an indented block」や「unexpected indent」などが表示されます。

## 例
以下は、`IndentationError`を引き起こす基本的な例です。

```python
def greet(name):
print("Hello, " + name)  # インデントが不足しているためエラー

greet("Alice")
```

修正後のコードは以下の通りです。

```python
def greet(name):
    print("Hello, " + name)  # 正しいインデント

greet("Alice")
```

## 説明
`IndentationError`を避けるための一般的な注意点は以下の通りです。

- **スペースとタブの混在**: インデントをスペースで行う場合は、すべての行でスペースを使用し、タブとは混合しないようにします。
- **一貫性の保持**: 同じブロック内では、すべての行が同じ深さのインデントを持つようにします。
- **エディタの設定**: 使用しているテキストエディタやIDEの設定で、インデントのスタイルを統一することができます。例えば、スペース4つでインデントする設定を行うなど。

## 一文要約
`IndentationError`は、Pythonにおいてインデントの不一致や不足が原因で発生するエラーです。
