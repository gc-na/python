# Python 中的 "complex" 數據類型：全面指南

## 簡介
在 Python 編程中，`complex` 類型用來表示複數。複數是一種數學概念，包含實數和虛數部分，通常以形式 `a + bj` 表示，其中 `a` 是實部，`b` 是虛部，`j` 是虛數單位。

## 文檔
### 目的
`complex` 類型在 Python 中提供了一種簡便的方法來處理複數數據。這對於數學運算、信號處理以及工程應用等領域非常重要。

### 用法
在 Python 中，可以使用內建的 `complex()` 函數來創建複數。可以通過兩種方式來實例化複數：
1. 使用 `complex(real, imag)`，其中 `real` 是實部，`imag` 是虛部。
2. 通過字面量表示法，例如 `1 + 2j`。

### 詳細說明
- **創建複數**：
  ```python
  z1 = complex(1, 2)  # 創建實部為 1，虛部為 2 的複數
  z2 = 3 + 4j         # 字面量表示法
  ```

- **複數屬性**：
  複數對象有兩個屬性：
  - `.real`：返回複數的實部。
  - `.imag`：返回複數的虛部。

- **運算**：
  Python 支持對複數的加、減、乘、除等基本運算。

## 示例
### 基本用法
```python
# 創建複數
z1 = complex(3, 4)
print(z1)  # 輸出： (3+4j)

# 獲取實部和虛部
print(z1.real)  # 輸出： 3.0
print(z1.imag)  # 輸出： 4.0

# 複數運算
z2 = complex(1, 2)
z3 = z1 + z2
print(z3)  # 輸出： (4+6j)

z4 = z1 * z2
print(z4)  # 輸出： (-5+10j)
```

## 解釋
使用 `complex` 類型時，有幾個常見的注意事項：
- 在 Python 中，虛部必須以 `j` 表示，而不是 `i`。
- 當使用浮點數創建複數時，實部和虛部將自動轉換為浮點數。
- 注意在進行複數運算時，結果也將是複數類型，即使實部或虛部為 0。

## 一句總結
`complex` 類型在 Python 中提供了強大的複數數據處理功能，適合數學及工程計算應用。
