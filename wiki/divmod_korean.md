# Python의 divmod 함수: 나눗셈과 나머지 계산하기

## 개요
Python의 `divmod` 함수는 두 숫자를 나누고, 결과로 몫과 나머지를 튜플 형태로 반환하는 내장 함수입니다. 이 함수는 나눗셈 연산을 간결하게 수행할 수 있도록 도와줍니다.

## 문서화
### 목적
`divmod` 함수는 주어진 두 숫자에 대해 몫과 나머지를 동시에 계산하여 반환합니다. 이를 통해 두 개의 연산을 하나의 함수 호출로 간단하게 수행할 수 있습니다.

### 사용법
`divmod` 함수의 기본 구문은 다음과 같습니다:

```python
divmod(a, b)
```

- `a`: 나누어질 숫자 (피제수)
- `b`: 나누는 숫자 (제수)

이 함수는 `(몫, 나머지)` 형태의 튜플을 반환합니다.

### 세부사항
- `a`와 `b` 모두 정수 또는 실수일 수 있습니다.
- `b`가 0일 경우 `ZeroDivisionError`가 발생합니다.
- 반환된 튜플의 첫 번째 요소는 몫이고, 두 번째 요소는 나머지입니다.

## 예제
### 기본 사용 예제
```python
# 두 숫자에 대한 divmod 사용
result = divmod(10, 3)
print(result)  # 출력: (3, 1)
```

### 음수와 실수 사용 예제
```python
# 음수와 실수에 대한 divmod 사용
result1 = divmod(-10, 3)
print(result1)  # 출력: (-4, 2)

result2 = divmod(10.5, 2)
print(result2)  # 출력: (5.0, 0.5)
```

## 설명
- `divmod`를 사용할 때의 일반적인 실수 중 하나는 제수가 0인 경우입니다. 이 경우 `ZeroDivisionError`가 발생하므로 항상 제수가 0이 아닌지 확인해야 합니다.
- 또한, `divmod`는 음수와 실수를 잘 처리하지만, 결과가 예상과 다를 수 있으므로 항상 주의해야 합니다.

## 한 줄 요약
`divmod` 함수는 두 숫자에 대해 몫과 나머지를 동시에 계산하여 튜플로 반환하는 유용한 Python 내장 함수입니다.
