# Python의 float: 부동 소수점 숫자 다루기

## 개요
Python의 `float`는 부동 소수점 숫자를 표현하는 자료형으로, 실수를 다루는 데 사용됩니다. `float` 자료형은 소수점 이하의 값을 포함할 수 있으며, 과학적 표기법을 통해 매우 큰 수나 작은 수를 다룰 수 있습니다.

## 문서화

### 목적
`float` 자료형은 숫자 계산 및 데이터 분석에서 실수를 표현하는 데 필수적입니다. 이는 금융, 과학, 공학 등 다양한 분야에서 중요한 역할을 합니다.

### 사용법
Python에서 `float`는 기본적으로 소수점 숫자를 생성할 때 사용됩니다. `float`는 다음과 같이 선언할 수 있습니다:

```python
num = 3.14          # 일반적인 소수 표현
num_scientific = 2.5e3  # 과학적 표기법 사용
```

이외에도 `float` 함수는 다른 자료형을 부동 소수점 숫자로 변환할 때 사용됩니다:

```python
int_num = 5
float_num = float(int_num)  # 정수를 부동 소수점으로 변환
```

### 세부 사항
- Python의 `float`는 64비트 배정밀도(IEEE 754)를 기반으로 하며, 이는 매우 큰 수와 작은 수를 정확하게 표현합니다.
- `float` 값은 사칙연산, 비교 등 다양한 수학적 연산에 사용할 수 있습니다.
- Python에서 `float`는 `Decimal` 자료형과 함께 사용하여 보다 정밀한 계산을 요구하는 상황에서 활용될 수 있습니다.

## 예제

### 기본 사용 예
```python
# 간단한 float 선언
a = 10.5
b = 2.0

# 덧셈
result = a + b  # 12.5
print(result)

# 과학적 표기법
c = 1.5e-4  # 0.00015
print(c)
```

### 변환 예
```python
# 다른 자료형을 float로 변환
string_num = "3.14"
converted_num = float(string_num)  # 3.14로 변환
print(converted_num)
```

## 설명
- **정확도 손실**: `float`는 소수점 이하의 특정 숫자를 정확하게 표현하지 못할 수 있습니다. 예를 들어, `0.1 + 0.2`의 결과는 `0.30000000000000004`가 될 수 있습니다. 이는 부동 소수점 연산의 특성으로, 정밀한 계산이 필요한 경우 `Decimal`을 사용하는 것이 좋습니다.
- **NaN과 Inf**: `float`는 Not-a-Number(NaN)과 무한대(Inf) 값도 지원합니다. 이는 특정 계산에서 발생할 수 있는 특별한 경우를 처리하는 데 유용합니다.
  
## 한 줄 요약
Python의 `float`는 부동 소수점 숫자를 표현하며, 실수 계산에 필수적인 자료형입니다.
