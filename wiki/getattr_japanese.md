# Pythonのgetattr関数: 属性の動的取得

## 概要
`getattr`は、Pythonオブジェクトから指定した属性を動的に取得するための組み込み関数です。この関数を使用することで、オブジェクトの属性にアクセスする際に、名前を文字列として渡すことができます。

## ドキュメンテーション
`getattr`関数は、以下のシンタックスで使用します。

```python
getattr(object, name[, default])
```

### パラメータ
- **object**: 属性を取得したい対象のオブジェクト。
- **name**: 取得したい属性の名前を示す文字列。
- **default**: （オプション）指定した属性が存在しない場合に返される値。デフォルトは`AttributeError`を投げます。

### 目的
`getattr`は、オブジェクトの属性を動的にアクセスするために使用され、特に属性名がプログラムの実行時に決定される場合に便利です。

### 使用法
`getattr`を利用することで、属性の存在を事前に確認せずにその値を取得できます。また、デフォルト値を指定することで、エラーを回避することも可能です。

## 例
以下に`getattr`の基本的な使用例を示します。

```python
class Sample:
    def __init__(self):
        self.name = "Python"

obj = Sample()

# 属性を取得
print(getattr(obj, 'name'))  # 出力: Python

# 存在しない属性を取得しようとする
print(getattr(obj, 'age', '属性が存在しません'))  # 出力: 属性が存在しません
```

## 説明
`getattr`を使う際の注意点：
- 属性がオブジェクトに存在しない場合、デフォルト値を指定しないと`AttributeError`が発生します。そのため、デフォルト値を設定することが一般的です。
- 属性名に対して誤った文字列を指定すると、期待した結果が得られません。正しい名前を使用することが重要です。
- `getattr`は組み込みの関数であり、ユーザー定義のクラスにも対応していますが、クラスが属性を動的に生成する場合は注意が必要です。
