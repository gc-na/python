# تنسيق السلاسل النصية في بايثون: استخدام الدالة format

## الملخص
تعتبر دالة `format` في بايثون أداة قوية ومرنة لتنسيق السلاسل النصية، مما يسهل إدراج المتغيرات في النصوص بطريقة منظمة وواضحة.

## الوثائق
تستخدم دالة `format` لتنسيق السلاسل النصية عن طريق إدراج القيم المتغيرة في أماكن محددة ضمن نص ثابت. يمكن استخدام هذه الدالة مع الأقواس المعقوفة `{}` حيث يتم استبدالها بالقيم الممررة إلى الدالة. هذا يمكن أن يكون مفيدًا لإنشاء نصوص ديناميكية.

### الاستخدام
يمكن استخدام `format` بطرق مختلفة:
1. **التنسيق البسيط**:
   ```python
   name = "أحمد"
   greeting = "مرحبا، {}!".format(name)
   print(greeting)  # يظهر: مرحبا، أحمد!
   ```

2. **التنسيق مع مواضع محددة**:
   ```python
   first_name = "أحمد"
   last_name = "علي"
   full_name = "{} {}".format(first_name, last_name)
   print(full_name)  # يظهر: أحمد علي
   ```

3. **التنسيق مع قيم محددة**:
   ```python
   age = 25
   info = "عمري {} سنة".format(age)
   print(info)  # يظهر: عمري 25 سنة
   ```

### التفاصيل
- **المعاملات**: يمكن تمرير عدد غير محدود من المعاملات إلى دالة `format`.
- **التنسيق المتقدم**: يمكن استخدام التنسيق المتقدم مثل تحديد دقة الأرقام أو تنسيق التواريخ داخل الأقواس.
- **التوافق**: دالة `format` متوافقة مع جميع إصدارات بايثون 2.7 وما فوق.

## الأمثلة
### مثال 1: تنسيق بسيط
```python
item = "تفاح"
quantity = 5
message = "لديك {} من {}".format(quantity, item)
print(message)  # يظهر: لديك 5 من تفاح
```

### مثال 2: استخدام الفهارس
```python
template = "الرقم الأول هو {0} والرقم الثاني هو {1}"
formatted = template.format(10, 20)
print(formatted)  # يظهر: الرقم الأول هو 10 والرقم الثاني هو 20
```

### مثال 3: تنسيق الأرقام
```python
price = 49.99
formatted_price = "سعر المنتج هو {:.2f} درهم".format(price)
print(formatted_price)  # يظهر: سعر المنتج هو 49.99 درهم
```

## الشرح
على الرغم من قوة دالة `format`، قد يواجه المستخدم بعض التحديات:
- **تناسق الأقواس**: يجب التأكد من أن عدد الأقواس المعقوفة `{}` يتطابق مع عدد القيم الممررة إلى الدالة.
- **الأداء**: في بعض الحالات، قد تكون دالة `f-string` (المقدمة في بايثون 3.6) أكثر كفاءة وسهولة في الاستخدام.

## ملخص جملة واحدة
دالة `format` في بايثون توفر وسيلة مرنة وفعالة لتنسيق السلاسل النصية وإدراج القيم المتغيرة فيها.
