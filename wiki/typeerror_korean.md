# TypeError: 파이썬에서의 타입 오류

## 개요
TypeError는 파이썬 프로그래밍에서 데이터 타입 간의 불일치로 인해 발생하는 오류입니다. 이는 변수나 객체가 기대되는 타입이 아닐 때 발생하며, 프로그램의 실행을 중단시킵니다.

## 문서화
TypeError는 파이썬의 내장 예외 중 하나로, 주로 함수나 연산자가 잘못된 데이터 타입을 받을 때 발생합니다. 예를 들어, 숫자와 문자열을 더하려고 하거나 리스트에 정수를 추가하려고 할 때 이 오류가 발생할 수 있습니다. TypeError는 개발자가 코드에서 잘못된 타입을 사용하고 있음을 알려주는 중요한 오류 메시지입니다.

### 목적
TypeError는 프로그래머가 코드에서 잘못된 데이터 타입을 사용하고 있음을 경고하여, 이를 수정할 수 있도록 돕습니다.

### 사용법
TypeError는 파이썬에서 자동으로 발생하며, 사용자가 별도로 호출할 필요가 없습니다. 주로 다음과 같은 상황에서 발생합니다:
- 서로 다른 데이터 타입 간의 연산 시
- 잘못된 인자를 가진 함수 호출 시
- 예상되는 타입이 아닌 객체에 접근 시

## 예제
1. **잘못된 연산**:
   ```python
   result = 'Hello' + 5  # TypeError 발생
   ```

2. **잘못된 함수 인자**:
   ```python
   def add_numbers(a, b):
       return a + b

   add_numbers(5, '10')  # TypeError 발생
   ```

3. **잘못된 리스트 접근**:
   ```python
   my_list = [1, 2, 3]
   my_list[0] = 'a'  # 정상 작동
   my_list.append(4.5)  # 정상 작동
   my_list.append('string')  # 정상 작동
   print(my_list)  # [1, 2, 3, 4.5, 'string']
   ```

## 설명
TypeError는 다양한 이유로 발생할 수 있으며, 몇 가지 일반적인 상황은 다음과 같습니다:
- **잘못된 타입 간의 연산**: 숫자와 문자열을 더하려고 하거나 리스트와 정수를 곱하려고 할 때 발생합니다.
- **부적절한 함수 인자**: 함수가 특정 타입의 인자를 기대할 때, 다른 타입의 인자를 전달하면 TypeError가 발생합니다.
- **타입 변환 실패**: str() 함수를 사용하여 숫자를 문자열로 변환하는 경우, 숫자 외의 객체를 변환하려고 할 때 오류가 발생할 수 있습니다.

### 일반적인 실수
- 타입을 혼동하여 연산을 시도하는 경우
- 함수의 인자 타입을 확인하지 않고 호출하는 경우
- 리스트에 예상치 못한 타입의 객체를 추가하는 경우

## 요약
TypeError는 데이터 타입 간의 불일치로 인해 발생하는 오류로, 프로그래머가 코드에서 올바른 데이터 타입을 사용하도록 경고하는 중요한 역할을 합니다.
