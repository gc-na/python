# ASCII في بايثون: التعامل مع النصوص والرموز

## ملخص
تعتبر ASCII (American Standard Code for Information Interchange) واحدة من أهم رموز الترميز المستخدمة في برمجة الحاسوب، وخاصة في لغة بايثون. يتيح لك استخدام ASCII التعامل مع النصوص والرموز بشكل فعال.

## الوثائق
تُستخدم ASCII لتمثيل الأحرف والأرقام والعلامات الخاصة في النظام الرقمي. في بايثون، يمكن استخدام الدالة `ord()` للحصول على قيمة ASCII لحرف معين، والدالة `chr()` لتحويل قيمة ASCII إلى الحرف المقابل لها.

### الاستخدامات:
- **التحويل بين الرموز وقيمها**: يمكنك استخدام `ord()` لتحويل الحرف إلى قيمته الرقمية و`chr()` للقيام بالعكس.
- **التأكد من كون الحرف ASCII**: يمكن استخدام دالة `isascii()` (المتوفرة في بايثون 3.7 وما بعدها) للتحقق مما إذا كان النص يتكون بالكامل من أحرف ASCII.

## أمثلة
### مثال 1: استخدام `ord()`
```python
# الحصول على قيمة ASCII لحرف 'A'
ascii_value = ord('A')
print(ascii_value)  # الناتج: 65
```

### مثال 2: استخدام `chr()`
```python
# تحويل قيمة ASCII إلى حرف
character = chr(65)
print(character)  # الناتج: 'A'
```

### مثال 3: استخدام `isascii()`
```python
# التحقق مما إذا كان النص يتكون من أحرف ASCII
text = "Hello"
is_ascii = text.isascii()
print(is_ascii)  # الناتج: True
```

## الشرح
### النقاط الشائعة:
- **قيمة ASCII**: القيم تتراوح من 0 إلى 127، حيث تشمل الأحرف الكبيرة والصغيرة، الأرقام، وبعض الرموز الخاصة. 
- **التأكد من النص**: إذا كان النص يحتوي على أحرف غير ASCII، ستعيد دالة `isascii()` القيمة False.
- **الإصدارات السابقة**: تأكد من أن تستخدم بايثون 3.7 أو أحدث عند استخدام `isascii()`.

## ملخص جملة واحدة
تعتبر ASCII في بايثون وسيلة فعالة لمعالجة الرموز والنصوص عبر الدوال `ord()` و`chr()` و`isascii()`.
