# Python의 dir() 함수: 객체의 속성과 메서드 탐색하기

## 개요
Python의 `dir()` 함수는 객체가 가지고 있는 속성(attribute)과 메서드(method)의 목록을 반환하는 유용한 도구입니다. 이 함수는 디버깅 및 개발 과정에서 객체의 구조를 이해하는 데 도움을 줍니다.

## 문서화
### 목적
`dir()` 함수는 주어진 객체의 속성과 메서드를 나열하여, 사용자가 객체의 인터페이스를 쉽게 파악할 수 있도록 합니다. 특히, 모듈, 클래스, 인스턴스, 내장 객체 등 다양한 객체에서 유용하게 사용됩니다.

### 사용법
`dir()` 함수는 다음과 같이 사용됩니다:

```python
dir([object])
```

- **object**: (선택 사항) 검토할 객체. 객체를 제공하지 않으면 현재의 로컬 심볼 테이블을 반환합니다.

### 세부 사항
- 반환값: `dir()`은 문자열의 리스트를 반환하며, 이 리스트는 객체의 속성과 메서드 이름을 포함합니다.
- 객체가 없는 경우: 인수를 제공하지 않으면, 현재 스코프의 모든 이름(변수, 함수 등)을 나열합니다.
- 내장 객체: 내장 타입의 객체에 대해 `dir()`을 호출하면, 클래스의 메서드와 속성도 포함됩니다.

## 예제
### 기본 사용 예
1. 기본 객체에 대한 `dir()` 사용:
   ```python
   my_list = [1, 2, 3]
   print(dir(my_list))
   ```

2. 사용자 정의 클래스에 대한 `dir()` 사용:
   ```python
   class MyClass:
       def __init__(self):
           self.attribute = 'value'
       def method(self):
           return 'method called'

   obj = MyClass()
   print(dir(obj))
   ```

3. 현재 스코프의 이름 나열:
   ```python
   a = 10
   b = 'Hello'
   print(dir())
   ```

## 설명
- **일관성**: `dir()`의 결과는 항상 동적으로 변화할 수 있습니다. 객체의 상태나 속성이 변경되면, `dir()`의 결과도 바뀝니다.
- **내부 속성**: `dir()`은 객체의 내부 속성(예: `__init__`, `__str__` 등)도 나열합니다. 이러한 속성은 언더스코어로 시작하므로, 주의하여 사용할 필요가 있습니다.
- **가독성**: `dir()`의 결과는 기본적으로 알파벳 순서로 정렬되어 있어, 쉽게 정보를 찾아볼 수 있습니다.

## 한 줄 요약
`dir()` 함수는 Python 객체의 속성과 메서드를 나열하여, 객체의 구조를 쉽게 이해하게 해주는 도구입니다.
