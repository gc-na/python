# Python 的 abs() 函數：獲取數字的絕對值

## 概述
`abs()` 是 Python 的一個內建函數，用於返回一個數字的絕對值。無論輸入的數值是正數、負數，還是零，`abs()` 都能提供一個非負的數值作為結果。

## 文件說明
`abs()` 函數的主要目的是計算給定數字的絕對值，這在數學運算中相當常見。該函數接受一個參數，並可處理整數、浮點數以及其他可轉換為數字的類型（如複數）。

### 使用方法
```python
abs(x)
```

- **參數**：
  - `x`：可以是整數、浮點數或複數。若為複數，返回的是其模（即絕對值）。

- **返回值**：
  - 返回 `x` 的絕對值，類型與輸入相同（整數或浮點數），或返回複數的模。

## 例子
### 整數的絕對值
```python
print(abs(-5))  # 輸出: 5
```

### 浮點數的絕對值
```python
print(abs(-3.14))  # 輸出: 3.14
```

### 複數的模
```python
print(abs(3 - 4j))  # 輸出: 5.0
```

## 解說
在使用 `abs()` 函數時，要注意以下幾點：
- 對於整數和浮點數，`abs()` 總是返回非負值。
- 對於複數，`abs()` 返回的是複數的模，計算方法是 `sqrt(real^2 + imag^2)`。
- 在使用浮點數時，可能會遇到精度問題，這在數學計算中是常見的，但對於 `abs()` 函數本身不會造成影響。

## 一句總結
`abs()` 函數用於計算數字的絕對值，無論輸入為正數、負數還是複數。
