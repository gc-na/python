# Python 中的 "all" 函數：完美檢查所有條件的工具

## 概述
在 Python 中，`all()` 是一個內建函數，用於檢查可迭代對象中的所有元素是否都為真值。當所有元素均為真時，返回 `True`，否則返回 `False`。這個函數在進行條件檢查時非常有用，特別是在需要確保所有條件均滿足的情況下。

## 文檔
### 目的
`all()` 函數的主要目的是提供一種方便的方法來檢查可迭代對象中的每個元素是否都為真值。

### 語法
```python
all(iterable)
```

### 參數
- **iterable**: 可以是任何可迭代對象，例如列表、元組、集合等。

### 返回值
- 返回 `True` 如果所有元素為真值（即非零、非空、非False），如果 iterable 為空，則返回 `True`。否則返回 `False`。

## 示例
### 基本用法
```python
# 示例 1: 檢查列表中的元素
result1 = all([True, True, True])  # 返回 True
result2 = all([True, False, True])  # 返回 False

# 示例 2: 檢查數字列表
numbers = [1, 2, 3, 4]
result3 = all(num > 0 for num in numbers)  # 返回 True

# 示例 3: 空可迭代對象
result4 = all([])  # 返回 True
```

## 解釋
使用 `all()` 時，有一些常見的注意事項：
- 空的可迭代對象會返回 `True`，這是因為在邏輯上，"所有都為真"的說法在無元素的情況下是成立的。
- `all()` 函數會對可迭代對象的元素進行短路評估，這意味著一旦找到第一個假值，函數就會立即返回 `False`，不會檢查後續的元素。
- 在使用生成器表達式時，`all()` 非常高效，因為它不會創建整個列表，而是逐個評估元素。

## 一句總結
`all()` 是一個方便的 Python 內建函數，用於檢查所有可迭代對象中的元素是否均為真值，返回布爾值。
