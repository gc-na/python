# Pythonにおける「True」の解説

## 概要
Pythonにおける「True」は、ブール値の一つであり、論理的な真を表現します。条件文や論理演算、比較演算において重要な役割を果たします。

## ドキュメンテーション
「True」はPythonの組み込み定数で、Boolean型の値を持ちます。この値は、0以外の数値や非空のオブジェクトと同様に、条件文で「真」として評価されます。

### 目的
- 条件判断における真偽を示す。
- 論理演算（AND、OR、NOT）において使用される。

### 使用法
`True`は、条件文やループ、比較演算子などで使用されます。以下はその主な用途です。
- 条件文での評価
- 論理演算におけるフラグ
- データ構造における状態管理

## 例
```python
# 条件文での使用
if True:
    print("これは真です")

# 比較演算
a = 5
b = 3
print(a > b)  # 出力: True

# 論理演算での使用
result = True and False
print(result)  # 出力: False
```

## 説明
「True」はPythonにおける特別な値であり、Pythonの他のデータ型と同様に扱われます。ただし、注意が必要な点もあります。

### よくある落とし穴
- 大文字で「True」と書く必要があります。小文字の「true」はエラーになります。
- 数値の1も「真」として評価されますが、明示的に「True」とすることでコードの可読性が向上します。
- 反対に、数値の0は「False」と評価されます。

## 一文の要約
Pythonにおける「True」は、条件判断や論理演算において真を表す重要な値です。
