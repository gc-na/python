# الفلتر في بايثون: كيفية استخدام الدالة filter لتصفية البيانات

## ملخص
تعد دالة `filter` في بايثون أداة قوية تُستخدم لتصفية العناصر من قائمة أو أي كائن قابل للتكرار بناءً على شرط معين (دالة). تُعيد الدالة عناصر جديدة تلبي الشرط المحدد.

## الوثائق
### الغرض
تُستخدم دالة `filter` لتصفية البيانات، حيث تأخذ دالة و iterable (مثل قائمة أو مجموعة) وتُرجع عناصر iterable التي تعود عليها الدالة بقيمة True.

### الاستخدام
تتبع دالة `filter` الصيغة التالية:
```python
filter(function, iterable)
```
- **function**: دالة تُطبق على كل عنصر في iterable. يجب أن تُعيد True أو False.
- **iterable**: كائن يمكن تكراره مثل قائمة، مجموعة، أو سلسلة.

### التفاصيل
- إذا كانت `function` None، ستقوم `filter` بإرجاع جميع العناصر التي تُعتبر True.
- يتم إرجاع كائن من نوع `filter`، والذي يمكن تحويله إلى قائمة باستخدام `list()`.

## الأمثلة
### مثال 1: تصفية قائمة الأعداد
```python
def is_even(num):
    return num % 2 == 0

numbers = [1, 2, 3, 4, 5, 6]
even_numbers = filter(is_even, numbers)
print(list(even_numbers))  # الناتج: [2, 4, 6]
```

### مثال 2: استخدام دالة lambda
```python
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = filter(lambda x: x % 2 == 0, numbers)
print(list(even_numbers))  # الناتج: [2, 4, 6]
```

### مثال 3: تصفية سلسلة من الأحرف
```python
words = ['apple', 'banana', 'cherry', 'date']
filtered_words = filter(lambda word: len(word) > 5, words)
print(list(filtered_words))  # الناتج: ['banana', 'cherry']
```

## الشرح
### النقاط الشائعة
- **الإرجاع من دالة filter**: يجب أن تكون دالة `function` واضحة في إرجاع قيم True أو False، وإلا فقد تؤدي إلى نتائج غير متوقعة.
- **النتيجة ككائن filter**: يجب تحويل الناتج إلى قائمة أو مجموعة إذا كنت ترغب في استخدامه مباشرة، حيث أن كائن `filter` نفسه لا يظهر العناصر بشكل مباشر.

### ملاحظات إضافية
- يمكن استخدام `filter` مع كائنات أخرى مثل المجموعات (sets) أو القوائم المرتبطة (tuples) ولكن تُفضل القوائم عادةً.
- تأكد من أن الدالة المستخدمة في `filter` تُعالج جميع أنواع البيانات التي قد توجد في iterable.

## ملخص جملة واحدة
تُستخدم دالة `filter` في بايثون لتصفية العناصر من iterable بناءً على شرط معين يُحدد بواسطة دالة.
