# Python에서의 True: 참(True) 값 이해하기

## 개요
Python에서 `True`는 불리언 데이터 타입의 진리 값으로, 조건문 및 논리 연산에서 사용되며, 프로그램의 흐름 제어에 중요한 역할을 합니다.

## 문서화
`True`는 Python의 내장 상수 중 하나로, 논리적 참을 나타냅니다. 불리언 데이터 타입은 `True`와 `False` 두 가지 값을 가지며, 조건문, 반복문, 그리고 다양한 논리 연산에 사용됩니다. `True`는 조건이 참일 경우에만 실행되는 코드 블록을 정의하는 데 중요한 역할을 합니다.

### 목적
- 조건문의 판단 기준 제공
- 논리 연산 수행
- 데이터의 참/거짓 여부 확인

### 사용법
`True`는 코드 내에서 직접 사용되며, 다음과 같은 구문에서 활용됩니다:
```python
if condition:
    # condition이 True일 때 실행되는 코드
```

## 예제
다음은 `True`를 사용하는 기본적인 예제입니다.

### 예제 1: 조건문에서의 사용
```python
x = 5
if x > 0:
    print("x는 양수입니다.")  # 이 코드 블록은 실행됩니다.
```

### 예제 2: 논리 연산에서의 사용
```python
a = True
b = False
result = a and b  # result는 False입니다.
```

### 예제 3: 반복문에서의 사용
```python
count = 0
while True:
    count += 1
    if count == 3:
        break  # count가 3이 되면 반복문 종료
print(count)  # 3이 출력됩니다.
```

## 설명
`True`는 불리언 값 중 하나로, 명시적으로 `True`로 설정된 변수는 조건문에서 참으로 평가됩니다. 하지만 `True`와 함께 사용할 때 주의해야 할 점은 아래와 같습니다.

- **대소문자 구분**: `true`와 `True`는 다릅니다. Python에서는 `True`만 올바른 값입니다.
- **유형 변환**: 다른 데이터 타입(예: 정수, 문자열 등)은 조건문에서 참 또는 거짓으로 변환됩니다. 예를 들어, `0`, `None`, 빈 문자열(`""`) 등은 `False`로 평가됩니다.

## 한 줄 요약
Python에서 `True`는 불리언 데이터 타입으로, 조건문의 참을 나타내는 내장 상수입니다.
