# Python의 dict: 파이썬 딕셔너리 완벽 가이드

## 개요
Python의 `dict`는 키-값 쌍을 저장하는 데이터 구조로, 빠른 데이터 조회와 수정이 가능합니다. 다양한 프로그래밍 상황에서 유용하게 사용되며, 데이터의 관계를 표현하는 데 강력한 도구입니다.

## 문서화
### 목적
`dict`는 파이썬에서 가장 많이 사용되는 내장 데이터 타입 중 하나로, 해시 테이블을 기반으로 구현되어 있습니다. 데이터의 키를 사용하여 값에 빠르게 접근할 수 있습니다.

### 사용법
딕셔너리는 중괄호 `{}`를 사용하여 생성하며, 각 키와 값은 콜론 `:`으로 구분됩니다. 키는 불변(immutable)해야 하며, 값은 어떠한 데이터 타입도 가능합니다.

예시:
```python
my_dict = {
    'name': 'Alice',
    'age': 30,
    'city': 'Seoul'
}
```

딕셔너리에서 값을 가져오려면, 키를 사용하여 접근합니다:
```python
print(my_dict['name'])  # 출력: Alice
```

### 상세 정보
- **키 중복**: 딕셔너리는 동일한 키를 두 번 사용할 수 없으며, 두 번째 값이 첫 번째 값을 덮어씁니다.
- **변경 가능성**: 딕셔너리는 변경 가능한 데이터 타입으로, 기존의 키-값 쌍을 수정하거나 새로운 쌍을 추가할 수 있습니다.
- **메서드**: `dict`는 다양한 내장 메서드를 제공합니다. 예를 들어, `.get()`, `.keys()`, `.values()`, `.items()` 등의 메서드를 사용하여 데이터를 다룰 수 있습니다.

## 예제
1. 딕셔너리 생성
```python
fruits = {
    'apple': 3,
    'banana': 5,
    'orange': 2
}
```

2. 값 접근
```python
print(fruits['banana'])  # 출력: 5
```

3. 값 추가
```python
fruits['grape'] = 4
```

4. 값 수정
```python
fruits['apple'] = 6
```

5. 값 삭제
```python
del fruits['orange']
```

6. 메서드 사용
```python
print(fruits.keys())   # 출력: dict_keys(['apple', 'banana', 'grape'])
print(fruits.values()) # 출력: dict_values([6, 5, 4])
print(fruits.items())  # 출력: dict_items([('apple', 6), ('banana', 5), ('grape', 4)])
```

## 설명
딕셔너리를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:
- **키의 불변성**: 리스트와 같은 가변 타입은 키로 사용할 수 없습니다. 대신 튜플이나 문자열과 같은 불변 타입을 사용해야 합니다.
- **키 존재 확인**: 키가 딕셔너리에 존재하지 않을 경우, `KeyError`가 발생합니다. `.get()` 메서드를 사용하면 기본값을 설정하여 이 문제를 피할 수 있습니다.
- **순서**: 파이썬 3.7 이후로, 딕셔너리는 삽입 순서를 기억합니다. 그러나 이전 버전에서는 순서가 보장되지 않았습니다.

## 한 줄 요약
Python의 `dict`는 키-값 쌍을 저장하여 빠른 데이터 접근을 가능하게 하는 강력한 내장 데이터 구조입니다.
