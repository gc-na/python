# Python의 복소수 (Complex Numbers) 이해하기

## 개요
Python에서 복소수(complex numbers)는 실수부와 허수부로 구성된 수학적 개념으로, `complex` 타입을 사용하여 표현됩니다. 이는 과학 및 공학 계산에서 매우 유용합니다.

## 문서화

### 목적
복소수는 수학 및 공학 문제를 해결할 때 필수적인 요소로, 특정 문제에서 실수만으로는 표현할 수 없는 수를 나타낼 수 있도록 합니다. Python은 복소수를 쉽게 사용할 수 있는 내장 지원을 제공합니다.

### 사용법
복소수를 생성하고 사용하는 방법은 다음과 같습니다:

- **복소수 생성**: `complex(real, imag)` 함수를 사용하거나 `a + bj` 형태로 직접 정의할 수 있습니다.
- **속성**: 복소수는 `.real` 속성과 `.imag` 속성을 통해 각각 실수부와 허수부에 접근할 수 있습니다.
- **연산**: 복소수 간의 덧셈, 뺄셈, 곱셈, 나눗셈과 같은 기본적인 수학 연산을 지원합니다.

### 상세 사용 예시
```python
# 복소수 생성
z1 = complex(2, 3)  # 2 + 3j
z2 = 4 + 5j         # 4 + 5j

# 복소수 속성 접근
print(z1.real)  # 2.0
print(z1.imag)  # 3.0

# 복소수 연산
z3 = z1 + z2    # 덧셈
z4 = z1 * z2    # 곱셈
print(z3)       # (6+8j)
print(z4)       # (-7+22j)
```

## 설명
복소수를 사용할 때 주의해야 할 점은 다음과 같습니다:

- **형 변환**: 복소수를 실수로 변환하려고 하면 오류가 발생합니다. 복소수는 항상 실수와 허수로 구성되어 있으므로, 이를 직접적으로 실수로 변환할 수는 없습니다.
- **허수부의 표현**: Python에서는 허수부를 `j`로 표기하며, 이는 수학적 표기법과 다릅니다. `i` 대신 `j`를 사용하는 점에 유의해야 합니다.
- **계산의 정확성**: 복소수를 사용할 때는 부동 소수점 연산의 특성으로 인해 발생할 수 있는 정확도 손실에 주의해야 합니다. 특히 복소수의 곱셈이나 나눗셈에서는 예상치 못한 결과가 나올 수 있습니다.

## 한 줄 요약
Python에서 복소수는 실수부와 허수부로 구성된 수학적 수로, 복잡한 계산을 간편하게 처리할 수 있도록 해줍니다.
