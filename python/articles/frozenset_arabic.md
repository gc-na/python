# مجموعة البيانات الثابتة (frozenset) في بايثون: كل ما تحتاج لمعرفته

## ملخص
تعتبر مجموعة البيانات الثابتة (frozenset) في بايثون نوعًا خاصًا من المجموعات، حيث تتميز بأنها غير قابلة للتغيير. تتيح لك هذه الميزة استخدام مجموعة لا تتغير بعد إنشائها، مما يجعلها مفيدة في تطبيقات عديدة، مثل المفاتيح في القواميس.

## الوثائق
تُستخدم مجموعة البيانات الثابتة (frozenset) لتخزين مجموعة من العناصر الفريدة، مثل المجموعات، ولكنها غير قابلة للتغيير. يُمكنك إنشاء مجموعة ثابتة باستخدام الدالة `frozenset()`، حيث يمكن أن تكون العناصر أية أنواع قابلة للتجزئة (hashable) مثل الأعداد الصحيحة، السلاسل النصية، أو حتى المجموعات الأخرى (لكن ليس المجموعات المتغيرة).

### الاستخدام
```python
# إنشاء مجموعة ثابتة
my_frozenset = frozenset([1, 2, 3, 4])
```

### الخصائص
1. **غير قابلة للتغيير**: لا يمكن تعديل عناصر مجموعة البيانات الثابتة بعد إنشائها.
2. **عناصر فريدة**: لا تُقبل العناصر المكررة.
3. **قابلة للتجزئة**: يمكن استخدامها كمفاتيح في القواميس.

## أمثلة
### مثال 1: إنشاء مجموعة ثابتة
```python
# إنشاء مجموعة ثابتة من قائمة
my_frozenset = frozenset([1, 2, 3, 3, 4])
print(my_frozenset)  # الناتج: frozenset({1, 2, 3, 4})
```

### مثال 2: استخدام مجموعة ثابتة كمفتاح في قاموس
```python
# استخدام مجموعة ثابتة كمفتاح في قاموس
my_dict = {frozenset([1, 2]): "قيمة 1", frozenset([3, 4]): "قيمة 2"}
print(my_dict[frozenset([1, 2])])  # الناتج: قيمة 1
```

### مثال 3: العمليات على المجموعات
```python
# العمليات على المجموعات
fs1 = frozenset([1, 2, 3])
fs2 = frozenset([3, 4, 5])
print(fs1.union(fs2))  # الناتج: frozenset({1, 2, 3, 4, 5})
print(fs1.intersection(fs2))  # الناتج: frozenset({3})
```

## الشرح
تُعتبر مجموعة البيانات الثابتة (frozenset) مثالية في الحالات التي تحتاج فيها إلى مجموعة من العناصر الفريدة التي لا ترغب في تعديلها بعد إنشائها. ومع ذلك، يجب أن تكون حذرًا عند استخدامها، حيث أن محاولة تعديل مجموعة ثابتة ستؤدي إلى رفع استثناء. 

### نقاط يجب الانتباه إليها
- لا يمكن إضافة أو إزالة عناصر من مجموعة ثابتة بعد إنشائها.
- لا يمكن استخدام المجموعات المتغيرة كعناصر داخل مجموعة ثابتة، لأنها غير قابلة للتجزئة.

## ملخص من سطر واحد
مجموعة البيانات الثابتة (frozenset) في بايثون هي نوع من المجموعات غير القابلة للتغيير التي تُستخدم لتخزين عناصر فريدة.
