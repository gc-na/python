# Python에서 NameError: 정의되지 않은 이름 오류

## 개요
Python에서 `NameError`는 코드에서 참조된 변수가 정의되지 않았거나 접근할 수 없는 경우 발생하는 오류입니다. 이 오류는 주로 변수의 오타, 선언되지 않은 변수 사용, 또는 특정 범위에서 변수를 참조할 때 발생합니다.

## 문서화
`NameError`는 Python의 내장 예외 중 하나로, 주로 프로그래밍 중에 변수나 함수 이름이 잘못 사용되었을 때 발생합니다. 이 오류는 코드의 가독성을 높이고, 디버깅 과정에서 변수와 함수의 정의를 확인할 수 있도록 돕습니다.

### 목적
`NameError`는 프로그래머가 잘못된 이름을 사용했음을 알리며, 코드의 흐름에서 변수의 정의와 접근 가능성을 확인하는 데 중요한 역할을 합니다.

### 사용법
`NameError`는 일반적으로 다음과 같은 경우에 발생합니다:
- 변수가 선언되지 않았거나 초기화되지 않은 경우
- 변수 이름의 오타가 있는 경우
- 함수나 클래스의 이름을 잘못 입력한 경우

## 예제
다음은 `NameError`가 발생하는 기본적인 예제입니다.

```python
# 예제 1: 선언되지 않은 변수 사용
print(x)  # NameError: name 'x' is not defined

# 예제 2: 오타로 인한 NameError
my_variable = 5
print(my_vriable)  # NameError: name 'my_vriable' is not defined

# 예제 3: 범위 외의 변수 참조
def my_function():
    print(y)  # NameError: name 'y' is not defined

my_function()
```

## 설명
`NameError`는 주로 다음과 같은 경우에 발생합니다:

1. **변수의 오타**: 변수를 잘못 입력하면 `NameError`가 발생합니다. 예를 들어, `my_variable`을 `my_vriable`로 잘못 입력하면 오류가 발생합니다.

2. **변수의 범위**: 변수가 정의된 범위를 벗어나서 접근하려고 하면 `NameError`가 발생합니다. 예를 들어, 함수 내에서 정의되지 않은 변수를 사용하면 오류가 발생합니다.

3. **선언되지 않은 변수**: 변수를 정의하지 않고 사용하면 `NameError`가 발생합니다. 이 경우 변수를 초기화해야 합니다.

### 추가 노트
- `NameError`는 디버깅 도구로서, 코드에서 변수가 올바르게 정의되고 사용되었는지 확인하는 데 도움이 됩니다.
- 이 오류는 프로그램의 실행을 중단시키므로, 발생한 위치를 정확히 찾아 수정하는 것이 중요합니다.

## 한 줄 요약
Python에서 `NameError`는 사용하려는 변수나 함수가 정의되지 않았거나 접근할 수 없을 때 발생하는 오류입니다.
