# Python 中的複數 (complex) 數據類型

## 概述
在 Python 中，`complex` 是一種內建數據類型，用於表示複數數字。複數由實部和虛部組成，虛部以字母 `j` 表示。這一數據類型對於數學計算、信號處理及工程應用等領域非常重要。

## 文件說明
`complex` 數據類型的主要目的是提供一種方便的方法來創建和操作複數數字。在 Python 中，你可以使用 `complex(real, imag)` 函數來創建複數，或者直接使用 `x + yj` 的語法。

### 用法
- **創建複數**:
  - 使用 `complex(real, imag)` 函數：
    ```python
    z1 = complex(2, 3)  # 實部為 2，虛部為 3
    ```
  - 使用字面量表示法：
    ```python
    z2 = 2 + 3j  # 同樣表示一個複數
    ```

- **屬性和方法**:
  - `z.real`：返回複數的實部。
  - `z.imag`：返回複數的虛部。
  - `abs(z)`：返回複數的模（大小）。
  - `conjugate(z)`：返回複數的共軛。

## 範例
以下是一些複數的基本用法範例：

```python
# 創建複數
z1 = complex(4, 5)
print(z1)  # 輸出： (4+5j)

# 使用字面量
z2 = 2 + 3j
print(z2)  # 輸出： (2+3j)

# 獲取實部和虛部
print(z1.real)  # 輸出： 4.0
print(z1.imag)  # 輸出： 5.0

# 計算模
print(abs(z1))  # 輸出： 6.4031242374328485

# 計算共軛
print(z1.conjugate())  # 輸出： (4-5j)
```

## 解釋
在使用複數時，有一些常見的陷阱和注意事項：
- **虛部標記**：虛部必須以 `j` 表示，而不是 `i`，這與數學習慣不同。
- **運算**：複數可以進行加、減、乘、除等運算，但結果仍然是複數。
- **類型轉換**：嘗試將複數直接轉換為整數或浮點數會引發錯誤，必須先提取實部或虛部。

## 一句總結
Python 的 `complex` 數據類型提供了一種簡便的方法來創建和操作複數，對數學和工程計算至關重要。
