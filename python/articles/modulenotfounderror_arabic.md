# ModuleNotFoundError في بايثون: سبب وحل مشكلة استيراد الوحدات

## ملخص
ModuleNotFoundError هو استثناء يُثار في بايثون عندما يفشل المفسر في العثور على وحدة (Module) محددة يتم استيرادها في البرنامج. هذا الاستثناء يُعتبر جزءًا من نظام معالجة الأخطاء في بايثون، ويُساعد المبرمجين في تحديد المشكلات المتعلقة بالاستيراد.

## الوثائق
### الغرض
ModuleNotFoundError يُستخدم للإشارة إلى عدم القدرة على العثور على وحدة معينة. يمكن أن يحدث ذلك عند محاولة استيراد وحدة غير موجودة، أو عندما لا تكون الوحدة موجودة في مسار البحث (PYTHONPATH).

### الاستخدام
يظهر هذا الاستثناء عادةً عند استخدام جملة `import`، مثل:
```python
import some_non_existent_module
```
عند تنفيذ الكود أعلاه، سيظهر الاستثناء بالشكل التالي:
```
ModuleNotFoundError: No module named 'some_non_existent_module'
```

### التفاصيل
- **الإصدار:** يُعتبر هذا الاستثناء جزءًا من بايثون 3.x، وقد تم استبدال استثناء ImportError به لتوفير مزيد من الوضوح.
- **السبب:** الأسباب الشائعة لظهور ModuleNotFoundError تشمل:
  - عدم تثبيت الوحدة المطلوبة.
  - وجود أخطاء في اسم الوحدة (مثل الأخطاء الإملائية).
  - عدم وجود الوحدة في مسار البحث (PYTHONPATH).
  - استخدام وحدات غير متوافقة مع الإصدار الحالي من بايثون.

## أمثلة
### مثال 1: استيراد وحدة غير موجودة
```python
try:
    import non_existent_module
except ModuleNotFoundError as e:
    print(f"خطأ: {e}")
```
### مثال 2: استيراد وحدة مثبتة ولكن في مسار خاطئ
```python
import sys
sys.path.append('/path/to/your/modules')
try:
    import your_module
except ModuleNotFoundError as e:
    print(f"خطأ: {e}")
```

## الشرح
### المشكلات الشائعة
- **الخطأ الإملائي:** تأكد من كتابة اسم الوحدة بشكل صحيح.
- **عدم وجود المكتبة:** تحقق من تثبيت المكتبة باستخدام `pip`:
    ```bash
    pip install your_module
    ```
- **مسار البحث (PYTHONPATH):** تأكد من أن المسار الذي يحتوي على الوحدة مضاف إلى PYTHONPATH:
    ```python
    import sys
    print(sys.path)
    ```

### ملاحظات إضافية
- في بعض الأحيان، قد تحتاج إلى إعادة تشغيل بيئة التطوير أو المحطة بعد تثبيت وحدة جديدة للتأكد من أن التغييرات سارية.
- تأكد من أنك تستخدم بيئة بايثون الصحيحة (مثل virtualenv أو conda) التي تحتوي على المكتبات المطلوبة.

## ملخص جملة واحدة
ModuleNotFoundError هو استثناء في بايثون يُشير إلى عدم القدرة على العثور على وحدة معينة أثناء عملية الاستيراد.
