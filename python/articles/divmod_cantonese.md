# Python 中的 divmod 函數：全面指南

## 概述
`divmod` 是 Python 中的一個內建函數，用於同時計算兩個數字的商和餘數。此函數對於需要同時獲取這兩個結果的情況特別有用，能夠提高程式的效率。

## 文檔
### 目的
`divmod` 函數的主要目的是簡化計算商和餘數的過程，這在處理整數運算時經常需要。它接受兩個數字作為參數，並返回一個包含商和餘數的元組。

### 使用方法
`divmod` 的語法如下：
```python
divmod(a, b)
```
- **參數**：
  - `a`：被除數，必須是整數或浮點數。
  - `b`：除數，必須是整數或浮點數，且不能為零。

- **返回值**：
  - 返回一個元組 `(商, 餘數)`，其中商是整數部分，餘數是除法運算後的剩餘部分。

### 詳細說明
- `divmod` 函數的運算遵循數學上的除法規則。
- 當 `a` 是正數且 `b` 是正數時，商為兩者的整數除法結果，餘數為 `a` 減去 `b` 乘以商的結果。
- 若 `a` 或 `b` 為負數，則根據 Python 的規則，餘數的符號會與除數 `b` 的符號相同。

## 範例
以下是一些 `divmod` 的基本使用範例：

### 範例 1：正整數
```python
result = divmod(10, 3)
print(result)  # 輸出 (3, 1)
```

### 範例 2：負數
```python
result = divmod(-10, 3)
print(result)  # 輸出 (-4, 2)
```

### 範例 3：浮點數
```python
result = divmod(10.5, 3)
print(result)  # 輸出 (3.0, 1.5)
```

## 說明
- **常見陷阱**：
  - 當除數 `b` 為零時，`divmod` 會引發 `ZeroDivisionError`。使用前應確保除數不為零。
  
- **額外注意**：
  - 使用 `divmod` 可以替代使用 `//` 和 `%` 兩個運算符，這樣不僅代碼更簡潔，還能提高可讀性。

## 總結
`divmod` 是一個高效的內建函數，用於同時獲取兩個數字的商和餘數，簡化了代碼並提高了運算效率。
