# Pythonの「divmod」関数：整数の商と剰余を求める

## 概要
Pythonの「divmod」関数は、2つの数値を引数として受け取り、その商と剰余をタプルとして返します。整数の除算を簡単に行うための便利な組み込み関数です。

## ドキュメンテーション
### 目的
「divmod」は、整数または浮動小数点数の商と剰余を一度の計算で求めるための関数です。これにより、計算を効率化し、コードを簡潔に保つことができます。

### 使用法
```python
divmod(a, b)
```
- `a`: 除数（整数または浮動小数点数）
- `b`: 被除数（整数または浮動小数点数）

この関数は、`a // b`（商）と`a % b`（剰余）をタプルとして返します。例えば、`divmod(10, 3)`を実行すると、結果は`(3, 1)`になります。

### 詳細
- `a`と`b`のどちらも数値でなければなりません。そうでない場合は`TypeError`が発生します。
- `b`がゼロの場合、`ZeroDivisionError`が発生します。
- 結果のタプルは、商と剰余の順で、`(商, 剰余)`となります。

## 例
以下は、`divmod`関数の基本的な使用例です。

```python
# 基本的な使用例
result = divmod(10, 3)
print(result)  # 出力: (3, 1)

# 負の数の使用例
result = divmod(-10, 3)
print(result)  # 出力: (-4, 2)

# 浮動小数点数の使用例
result = divmod(10.5, 3)
print(result)  # 出力: (3.0, 1.5)
```

## 説明
- **一般的な落とし穴**: `b`がゼロの場合、計算ができないため`ZeroDivisionError`が発生します。これを避けるためには、`b`がゼロでないかを事前に確認する必要があります。
- **整数と浮動小数点数の混合**: `divmod`は整数と浮動小数点数の両方を受け付けますが、結果は浮動小数点数になる場合があります。例えば、`divmod(10, 4.0)`は`(2.0, 2.0)`を返します。

## 一文要約
Pythonの`divmod`関数は、2つの数値の商と剰余を効率的に計算し、結果をタプルとして返す便利な組み込み関数です。
