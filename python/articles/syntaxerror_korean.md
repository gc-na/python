# Python의 SyntaxError: 문법 오류 이해하기

## 개요
Python에서의 SyntaxError는 코드가 파이썬 문법 규칙을 따르지 않을 때 발생하는 오류입니다. 이 오류는 프로그램이 실행되기 전에 발생하며, 코드가 올바른 형식이 아닐 경우 발생합니다.

## 문서화
### 목적
SyntaxError는 파이썬 인터프리터가 코드의 문법을 분석할 때 발생하는 오류입니다. 올바른 문법으로 작성되지 않은 코드를 실행하려고 하면 이 오류가 발생하여, 개발자가 코드를 수정할 수 있도록 돕습니다.

### 사용법
SyntaxError는 일반적으로 코드 작성 중 실수로 발생합니다. Python 코드를 작성할 때 다음과 같은 형식의 오류가 발생할 수 있습니다:
- 괄호, 중괄호, 대괄호의 짝이 맞지 않음
- 잘못된 들여쓰기
- 잘못된 구문 사용
- 예약어를 잘못 사용

Python 인터프리터는 SyntaxError가 발생하는 경우 오류 메시지를 출력하며, 오류가 발생한 줄 번호를 포함하여 개발자에게 문제의 위치를 알려줍니다.

### 세부사항
SyntaxError는 다음과 같은 상황에서 발생할 수 있습니다:
- 함수 정의에서 콜론(:) 누락
- 문자열을 닫지 않음
- 잘못된 변수 이름 사용
- 잘못된 연산자 사용

개발자는 이러한 오류 메시지를 통해 코드를 수정하고 올바른 문법으로 작성해야 합니다.

## 예제
다음은 SyntaxError가 발생하는 간단한 예제입니다.

```python
# 잘못된 예제: 닫는 괄호가 없음
print("Hello, World!"

# 잘못된 예제: 잘못된 들여쓰기
def my_function():
print("Hello!")

# 잘못된 예제: 잘못된 변수 이름
1variable = 10
```

위의 코드 조각들은 각각 SyntaxError를 발생시키며, 파이썬 인터프리터는 오류 메시지를 출력합니다.

## 설명
SyntaxError는 주로 초보 프로그래머가 자주 겪는 오류입니다. 다음은 이런 오류를 피하기 위한 팁입니다:
- 괄호와 중괄호의 짝을 항상 확인하세요.
- 문자열과 주석을 올바르게 닫는 것을 잊지 마세요.
- 코드의 들여쓰기를 일관되게 유지하세요.
- 예약어와 변수 이름의 사용에 주의하세요.

이러한 실수를 줄이면 SyntaxError를 예방하는 데 도움이 됩니다.

## 한 줄 요약
