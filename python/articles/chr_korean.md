# Python의 chr() 함수: 문자열과 아스키 코드 변환

## 개요
Python의 `chr()` 함수는 주어진 정수 값을 해당하는 유니코드 문자로 변환하는 기능을 제공합니다. 이 함수는 문자와 숫자 간의 변환을 간편하게 수행할 수 있도록 도와줍니다.

## 문서화
### 목적
`chr()` 함수는 주로 아스키 코드 또는 유니코드 코드 포인트를 문자로 변환하는 데 사용됩니다. 이 함수는 프로그래밍에서 문자 데이터를 처리할 때 유용하게 사용됩니다.

### 사용법
```python
chr(i)
```
- **매개변수**: 
  - `i`: 변환할 정수 값으로, 유니코드 코드 포인트를 나타냅니다. `0`에서 `1114111`(0x10FFFF) 사이의 값을 받아야 합니다.
  
- **반환값**: 
  - 주어진 정수에 해당하는 유니코드 문자를 반환합니다. 만약 잘못된 값이 들어올 경우 `ValueError`를 발생시킵니다.

### 세부사항
- `chr()` 함수는 Python 3.x 버전에서 사용되며, Python 2.x에서는 `unichr()` 함수가 사용되었습니다.
- 유니코드 코드 포인트는 각 문자에 고유하게 할당된 숫자입니다. 예를 들어, 'A'의 유니코드 코드 포인트는 `65`입니다.

## 예제
```python
# 예제 1: 아스키 코드 65를 문자로 변환
print(chr(65))  # 출력: A

# 예제 2: 아스키 코드 97을 문자로 변환
print(chr(97))  # 출력: a

# 예제 3: 유니코드 문자 변환
print(chr(9731))  # 출력: ☃ (눈사람)
```

## 설명
- `chr()` 함수를 사용할 때 주의해야 할 점은 인자로 입력되는 정수가 반드시 유효한 코드 포인트여야 한다는 것입니다. 예를 들어, `chr(-1)` 또는 `chr(1114112)`와 같이 범위를 벗어난 값을 입력하면 `ValueError`가 발생합니다.
- 또한, `chr()` 함수는 반대로 문자에서 코드 포인트로 변환하는 `ord()` 함수와 함께 사용되면 더욱 효과적입니다. `ord()` 함수는 주어진 문자의 유니코드 코드 포인트를 반환합니다.

## 한 줄 요약
Python의 `chr()` 함수는 정수 유니코드 코드 포인트를 해당하는 문자로 변환하는 유용한 도구입니다.
